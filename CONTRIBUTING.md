# Contributing to AcidGunner Bots

Thanks for your interest in helping! Here is how you can contribute:

1. **Fork the repo.**
2. **Create a branch:** `git checkout -b feature/your-feature-name`
3. **Commit your changes.**
4. **Push to your fork.**
5. **Open a Pull Request.**

### For Testers:
If you are testing the bots on Termux/Android, please report:
- Device model (e.g., Tecno Spark 20).
- Python version.
- Any error logs you see.
